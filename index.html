Projects — Cisco Systems

  • IEEE 802.1Qbu TSN Frame Preemption — Cisco Industrial Ethernet 9000
    Cisco Systems · 2024 – Present
    Time-Sensitive Networking requires high-priority frames to preempt in-flight non-critical traffic with microsecond-level latency guarantees — a capability that demands tight co-design across firmware, OS, and network stack. Implemented end-to-end TSN preemption support: platform firmware changes, QoS integration for handling express and preempted traffic queues, per-port CLI configuration, and an auto-enable mechanism that activates preemption only when both link endpoints negotiate support. Validated on Cisco IE9000 industrial switch platforms.
  • AI-Powered Automated Code Review Platform
    Cisco Systems · 2024 – Present
    At scale, manual code review misses subtle bugs, security issues, and cross-file regressions — especially when a change in one module silently breaks a dependency elsewhere. Built a system where specialized LLM agents independently analyze each PR for bugs, security, performance, and maintainability, grounded by retrieval over code knowledge graphs and semantic embeddings to reason across file boundaries. Agent findings are aggregated with static analysis signals and ranked by confidence; a developer feedback loop retrains the ranking model over time.
  • Graph RAG Agent for Network PHY/Port Debugging
    Cisco Systems · 2024 – Present
    PHY/port issues — link flaps, CRC bursts, SFP failures — rarely have a single-source root cause; diagnosis requires correlating CLI outputs, driver logs, transceiver metrics, and hardware errata across the full Port → PHY → SerDes → ASIC stack. Built a Graph RAG system that models these hardware dependencies as a knowledge graph, enabling an LLM agent to traverse causal paths and pinpoint likely failure points rather than returning semantically similar documents. Reduced average triage time from ~60 to ~5 minutes for complex PHY faults.
